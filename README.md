另外，如果你想要在你自己的电脑上搞，请项目根目录新建一个文件，名为`config.py`

内容如下：

```python
mode = 'dev'
smtp_server = 'smtp.xxx.com'
from_addr = 'xxx@xxx.com'
password = 'xxxxxx'
```

mode 表示是开发（dev）还是生产（prod），你也可以把他改成 prod
